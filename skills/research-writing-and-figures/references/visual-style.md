# Visual Style Reference

## Design goal

Scientific figures should look calm, legible, and intentional. Style must support the information, not compete with it.

## Panel logic

- Every panel must provide distinct information.
- If one panel only compresses or repeats what the others already show, remove it.
- Prefer fewer strong panels over crowded multi-panel figures.

## Color

- Use restrained, publication-style palettes.
- Prefer muted blue, gray-blue, clay, or teal families over saturated dashboard colors.
- Use one color plus marker or line-style variation where possible.
- De-emphasize helper lines, baselines, and reference elements.

## Labels and titles

- Keep panel labels consistent in size, weight, and relative position.
- Do not let panel titles repeat information already obvious from the caption or x-axis.
- Remove subplot titles when the caption and context already identify the setting.

## Caption relationship

- The caption explains how to read the figure.
- The main text explains why the figure matters.
- Do not make the figure title or caption carry the main scientific claim.

## Layout

- Avoid visual clutter.
- Give the main comparison the strongest contrast.
- Keep legends minimal and, when possible, shared across panels.

## Export and formatting

- Prefer vector output for manuscript figures.
- Keep typography and spacing consistent across all figures in the paper.
- Use the final intended size when judging legibility.

## Slide visuals

For talks, a visual should usually do one job quickly:

- establish the problem
- contrast two situations
- show one key mechanism

Do not overload a single slide with multiple competing ideas.
